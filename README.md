## ğŸš€ Quick Start

1. **Install the Plugin** (Choose one method):
    - Open the Obsidian **Community Plugins** market, search for **Fast Note Sync**, and install it.
    - Or, visit https://github.com/haierkeys/obsidian-fast-note-sync/releases to download the latest release package, and unzip it into your Obsidian plugin directory: `.obsidian/plugins`.
2. Open the plugin settings, click the **Paste Remote Config** button, and paste your remote service configuration string into the input box.
